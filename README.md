# KINGENIOUS WORKS | Beatmaker & Developer Portfolio

A high-performance, futuristic portfolio built with Next.js 15, Tailwind CSS, and Firebase.

## Features

- **Dynamic Content**: Edit text, images, and projects directly on the site (Admin Mode).
- **Audio Player**: persistent global beats player with waveform visualization.
- **Stealth Admin Access**: Secure, hidden login for the owner.
- **Glassmorphism Design**: High-end dark UI with neon accents.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Open** [http://localhost:3000](http://localhost:3000).

## Admin Access (Secret)

- **Desktop**: Press `Ctrl + Shift + L`.
- **Mobile**: Triple-tap the "KW" Logo in the Navbar.

## Deployment

Deployed on [Vercel](https://vercel.com).
Ensure strict environment variables are set for Cloudinary and Firebase.
